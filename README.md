# Игра «Найди пару» (Memory Card Game)

Классическая игра на запоминание, реализованная на чистом JavaScript с использованием ООП.

---

## О проекте

Это казуальная игра, где игроку нужно найти все пары одинаковых карточек. Проект демонстрирует мое понимание:
- Объектно-ориентированного программирования на JavaScript
- Управления состоянием приложения
- Асинхронных операций и работы с DOM
- Пользовательского опыта (UX)

**Ключевая особенность:** при завершении игры автоматически запускается новый раунд, позволяя играть бесконечно.

## Архитектура и решения

### 1. Объектно-ориентированный подход

Класс `Card` инкапсулирует всю логику отдельной карточки:
- Приватные поля `open` и `success` для отслеживания состояния
  
- Геттеры и сеттеры для контроля изменений
  
- Собственные методы управления DOM-элементом

```javascript
class Card {
    set isOpen(value) {
        this.open = value;
        value ? this.card.classList.add('open') : this.card.classList.remove('open');
    }
}
```

### 2. Управление игровой логикой

Основной модуль main.js отвечает за:
- Генерацию парных значений (каждая цифра встречается дважды)

- Перемешивание массива (Fisher-Yates через sort)

- Отслеживание выбранных карт и сравнение значений

- Обработку победы и рестарт игры

### 3. Пользовательский опыт

- Визуальная обратная связь: разные состояния карт (закрытая, открытая, угаданная)

- Плавные анимации: через CSS-переходы

- Адаптивность: сетка автоматически подстраивается под количество карт

- Бесконечный геймплей: после победы игра автоматически перезапускается с новым количеством карт

---

## Как это работает

1. Игрок вводит количество карт (четное число от 2 до 20)
2. Создается массив с парами чисел и перемешивается
3. Каждая карта получает обработчик клика
4. При клике карта открывается, система запоминает ее значение
5. После открытия второй карты происходит сравнение:
  5.1. Совпали → карты остаются открытыми (зеленый цвет)
  5.2. Не совпали → карты закрываются через мгновение
6.Когда все пары найдены — победа и новый раунд
